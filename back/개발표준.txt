2.  services 폴더에 서비스 모듈 생성
    2-1. 서비스 파일명은 기능명 + Service ex) usersService.js
    2-2. 메서드 명
    조회 : find + 범위 + 테이블명 ex) findAllCenter
    삽입 : add + 테이블명 ex) addUsers
    삭제 : remove + 범위 + 테이블명 ex) removeByIdUsers
    수정 : modify + 범위 + 테이블명 ex) modifyByIdUsers

    2-3. insert, update, delete할 때는 return할 때 아래와 같이 하기
    ex) insert의 경우
    let resObj = {};
    if (result.insertId > 0) {
    resObj = {
    status: "success",
    no: result.insertId,
    };
    } else {
    resObj = { status: "fail" };
    }

    return resObj;

    ex) update, delete의 경우
    let resObj = {};
    if (result.affectedRows > 0) {
    resObj = { status: "success", no: userNo };
    } else {
    resObj = { status: "fail" };
    }

    return resObj;
